# 📄 resume-optimizer

A simple browser-based tool to analyze resume-job matches using intelligent text analysis and provide actionable improvement suggestions. Just paste your job description and resume, then get instant insights.

## ✨ Features

* 📋 Paste job descriptions and resumes for instant analysis
* 🎯 Calculates match score using intelligent keyword analysis
* 📊 **Detailed analytics** with ATS score, readability, and keyword density
* 💪 **Strengths analysis** - highlights matching qualifications
* 🔍 **Gap analysis** - identifies missing requirements and improvement areas
* 💡 **Actionable suggestions** for resume optimization
* 🌗 **Dark/light mode** with auto-detect and toggle
* 📱 **Mobile-friendly** responsive design with modern UI
* ⚙️ Works entirely client-side — no backend required

## 🚀 Usage

1. Clone or fork this repo:

   ```bash
   git clone https://github.com/jajera/resume-optimizer
   cd resume-optimizer
   ```

2. Open `index.html` in a modern browser (Chrome, Edge, Safari).

3. Or [use the GitHub Pages version](https://jajera.github.io/resume-optimizer).

## 🧠 Tech Stack

* Vanilla HTML, CSS, JavaScript
* Advanced text analysis algorithms
* CSS Grid/Flexbox for responsive layouts
* Web APIs for theme detection and storage
